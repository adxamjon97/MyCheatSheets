# C++ dasturlash tili

Bu dasurlash tili C tilida kelib chiqqan.

g++ kompilyator o'rnatamiz

main.cpp fayl yaratamiz

```cpp
#include<iostream>
using namespace std;

int main(){
	cout<<"Hello world"<<endl;
	return 0;
}
```

yozamiz

g++ main.cpp

yoki

g++ main.cpp -o main

teramiz

chiquvchi fayl main (yoki a.out) bo'ladi

./main

yoki

./a.out
